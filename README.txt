This application was made by Corey Warden, Nipun Bhatnagar, and Ivan Ou Yang from Timberland High School, MO.
This program was made for the FBLA Computer Game & Simulation Competition and is a game designed for user to play through time in a series of classic games.
This program was made in Python and then converted to an executable file using pyinstaller.
Since the program was created and converted on a Windows 10, 64 bit computer, it will only work on such computers.
The main executable file, Genesis.exe, is located in the main folder.
